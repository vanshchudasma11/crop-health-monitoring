Low-cost AI-powered crop health monitoring system using multispectral imaging and machine learning (IIT Bombay e-Yantra project)

# 🚀 AI-Powered Crop Health Monitoring System  
### Low-Cost Multispectral Imaging for Smart Agriculture  

🚀 Actively improving and expanding this project.

---

## 🌍 Overview
This project focuses on building a **low-cost, scalable, and intelligent crop health monitoring system** using **multispectral imaging (RGB + NIR)** and **machine learning**.

The goal is to enable **early detection of plant stress and diseases**, which is critical for improving agricultural productivity and reducing losses.

---

## ❗ Problem
In India, farmers often detect crop issues only after visible symptoms appear, leading to significant yield loss.

Existing solutions like industrial multispectral cameras cost **₹1–3 lakh**, making them inaccessible to small and medium-scale farmers.

---

## ⚡ Solution
This system provides a **cost-efficient alternative (~₹9,000)** using:

- Recycled smartphone camera modules  
- NIR (Near-Infrared) imaging  
- NDVI-based plant health analysis  
- Machine learning classification  

---

## 🔥 Key Achievements
- 💸 Reduced system cost by **~92%** (₹1,00,000 → ₹9,000)  
- 🧠 Built ML pipeline using **Python + OpenCV**  
- 🌿 Classified plant health conditions (Healthy / Stressed / Diseased)  
- 📊 Developed real-time monitoring interface  
- ⚙️ Designed system to work **offline for rural use**  

---

## 🛠 Tech Stack
- **Python, OpenCV, NumPy**  
- **TensorFlow / Scikit-learn**  
- **ESP32-CAM / Raspberry Pi**  
- **IoT Sensors + NIR Filters**  
- **Web Interface (React)**  

---

## 🧠 System Workflow
1. Capture images (RGB + NIR)  
2. Perform reflectance calibration  
3. Compute NDVI / vegetation indices  
4. Apply ML model for classification  
5. Display results via dashboard  

---

## 🌱 Impact
Designed as a **scalable and affordable alternative** to expensive agricultural monitoring systems, enabling **precision farming for small-scale farmers**.
